arg1: the c program
arg2: the exe file (the file you run eventually for example ./run)
arg3: the times the exe file will run, 1 to (inf)

```
terminal
>> ./runprog.sh program.c run 1
arg1 = program.c
arg2 = ./run
arg3 = 1 to inf
```
